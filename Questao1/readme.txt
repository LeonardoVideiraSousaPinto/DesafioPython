Instruções para Utilização do Projeto da Questão 1

Clone ou baixe o repositório referente à questão 1.
Abra o terminal (ou shell).
Navegue até o diretório onde o projeto foi salvo usando o comando cd seguido pelo caminho para a pasta do projeto.
Execute o comando para iniciar o container composto, baixar todas as imagens necessárias e realizar a raspagem. Use docker-compose up --build.
O container será executado e gerará os arquivos json na pasta json do projeto.
Para parar e remover o container, execute o comando docker-compose down.
