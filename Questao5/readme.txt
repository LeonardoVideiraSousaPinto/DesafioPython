Este arquivo arvore.py mostra como criar e usar uma árvore binária em Python. Aqui está um resumo do que o código faz:


Classe Nodo:
Representa um nó na árvore. Cada nó tem um valor e dois filhos: esquerdo e direito.

Classe ArvoreBinaria:
Cria uma árvore binária com métodos para inserir valores e percorrer a árvore. Os principais métodos são:

inserir(valor): Adiciona um valor na árvore.
travessia_em_ordem(): Retorna uma lista com os valores em ordem crescente.

Testes
O código inclui testes para garantir que a árvore funcione corretamente:

Testa a inserção de vários valores e a travessia em ordem.
Testa a inserção de um único valor.
Testa a inserção de valores duplicados.

O Que É Uma Árvore Binária?
Uma árvore binária é uma estrutura onde cada nó pode ter até dois filhos: um à esquerda e um à direita. É útil para organizar dados de forma que você possa encontrar e armazenar informações rapidamente.
